# ShellCommand

Basic shell commands.

---

## Methods/Functions

| Method/Functions  | Description                                                       |
|-------------------|-------------------------------------------------------------------|
| StartApplication  | Starts up an application or script and passes optional arguments. |
| StartFileExplorer | Starts up the file explorer application.                          |

---

## Usage

```vb
Private Sub Demo()
    StartFileExplorer
End Sub
```
